Documentação tecnica

Projeto: AgricoAqui_api
Usando arquitetura MVC
Necessário ter node instalado
usando Node versão 20.19.6


npm install(instalar todas biblioteca do projeto)
ou 
Bibliotecas e frameworks separados:
- npm install Mysql2 - drive Myql para node
- npm install express- gerenciar rotas
- npm install sequelize- usado para gerencia banco de dado Mysql no Node
- npm install bcryptjs - criptografar senha no banco de dados

Antes de dar start no projeto certifique ser está conectado ao banco de dado na maquina
start: node server.js
Projeto roda porta
roda na porta 3800(http://localhost:3800)
